# Lahsa Tours - Premium Travel & Tours Website

A modern, premium React.js website for a travel and tours company featuring holiday packages, car rental services, and an intuitive search interface.

## Features

- **Banner Section**: Eye-catching hero banner with search form (From & To destinations)
- **Holiday Packages**: Showcase of premium travel packages with detailed information
- **Car Rental**: Premium vehicle rental service with fleet showcase
- **Responsive Design**: Fully responsive across all devices
- **Modern UI**: Professional and premium design using Tailwind CSS
- **Smooth Animations**: Engaging hover effects and transitions

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React (Icons)

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

2. **Banner**: Hero section with search form
3. **Search Form**: From & To destination search
4. **Holiday Packages**: Premium tour packages showcase
5. **Car Rental**: Vehicle rental service

## Customization

- Update colors in `tailwind.config.js`
- Modify package data in `HolidayPackages.jsx`
- Update car fleet in `CarRental.jsx`
- Customize banner content in `Banner.jsx`
